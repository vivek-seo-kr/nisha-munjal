## About

This portfolio is designed to help patients find and connect with Dr. Nisha Munjal online. It highlights her expertise in gynecology and fertility treatments, making it easy for patients to book appointments and get directions to her clinic in Kangra.

## Sections

- **Hero** — Introduction with specialization and call-to-action
- **About** — Doctor's background and 21+ years of experience
- **Services** — Gynecological and fertility treatments offered
- **Testimonials** — Patient reviews and success stories
- **FAQ** — Common queries about services and treatments
- **Contact** — Address, phone, and Google Maps location
- **Footer** — Social media links and clinic info

## Deployment

This project is deployed via **GitHub Pages**.

To deploy your own copy:

1. Fork or clone this repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site will be live at `https://your-username.github.io/repo-name`

## Local Preview

Just open `index.html` in any browser — no setup required.

```bash

cd your-repo-name

```
